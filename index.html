Claude + OpenClaw Integration

Connect Anthropic's Claude to your OpenClaw device for the best of both worlds: local autonomy with cloud intelligence when you need it.

Overview

OpenClaw supports Claude as a cloud backend. Your device handles automation, memory, and tool execution locally, while optionally routing complex reasoning tasks to Claude's API.

💡 Key Principle: Your data stays local. Only the specific prompt you send to Claude leaves your device.

Prerequisites

RequirementDetails
OpenClawv2.4+ installed on your device
Anthropic API keyconsole.anthropic.com
NetworkOutbound HTTPS to api.anthropic.com

Quick Setup

Edit your OpenClaw configuration:

# ~/.openclaw/config.yaml
providers:
  anthropic:
    api_key: "${ANTHROPIC_API_KEY}"
    default_model: claude-sonnet-4-20250514

Set your API key as an environment variable and restart:

$ export ANTHROPIC_API_KEY=sk-ant-...
$ openclaw restart
